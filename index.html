About

ico.js parse a *.ico file and create some *.bmp files.

To fully use this library, browsers must support JavaScript typed arrays and some Web API Interfaces below:

  • Blob
  • FileReader
  • XMLHttpRequest

Google Chrome, Internet Explorer 10, Mozilla Firefox and Safari 6 support these function.

To use this library in your HTML page, please include ico.js like this:

<script type="text/javascript" src="/path/to/ico.js"></script>

Documentation

ico.Parse

Parse ico file and return bitmap.

Usage: 

Array ico.Parse(ArrayBuffer buffer)

Parameters

NameTypeDescription
bufferArrayBufferThe ArrayBuffer object contain the TypedArray of a ico file.

Returns

ico.Parse returns bitmaps Array.

bitmaps.length means number of bitmap that was included in the ico file.

The bitmaps is as below:

bitmaps = [
    {
        bitmap: Uint8Array,
        width: Number,
        height: Number,
        bit: Number
    }
]
NameTypeDescription
bitmapUint8ArrayThe Uint8Array object consist of a bitmap file.
widthNumberPixel width of the bitmap.
heightNumberPixel height of the bitmap.
bitNumberColor depth of the bitmap.

Example

<input type="file" id="input-file" />
<script>
    document.getElementById('input-file').addEventListener('change', function (evt) {
        // use FileReader for converting File object to ArrayBuffer object
        var reader = new FileReader();
        reader.onload = function (e) {
            var bitmaps = ico.Parse(e.target.result);
            // logs bitmaps Array
            console.dir(bitmaps);
            for (var i = 0; i < bitmaps.length; i++) {
                var blob = new Blob([bitmaps[i].bitmap], { type: 'image/bmp' });
                var blobURL = window.URL.createObjectURL(blob);
                // logs object URL of a bitmap file
                console.log(blobURL);
            }
        };
        reader.readAsArrayBuffer(evt.target.files[0]);
    }, false);
</script>
